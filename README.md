# Tab Switch Shortcuts

Firefox extension that exposes customizable keyboard shortcuts for switching to the tab to the left or right while skipping over collapsed tab groups.

## Loading the Extension

- Open `about:debugging#/runtime/this-firefox`
- Click **Load Temporary Add-on…** and select `manifest.json` in `src/`

## Customizing Shortcuts

Go to `about:addons` → gear icon → **Manage Extension Shortcuts** and scroll to the entry for **Tab Switch Shortcuts**
